Racetrack

This project is a PC version of the classic Racetrack usually played with pen and paper. If you want to know more about the game and its rules, please visit this page at wikipedia.org dedicated to it.

This application allows you to:

  • Play the Racetrack game on different maps with any number of players.

  • Play against AI players.

  • Let multiple AIs play against each other.

  • Add your own AI to the game thanks to a simple interface.

  • Use Tiled map editor to create new maps for the game easily.
